# 📊 Dashboard de Vendas – Empresa de Esquadrias

## 🎯 Objetivo
Este projeto tem como objetivo analisar o desempenho de vendas de uma **empresa de pequeno porte do setor de esquadrias**, fornecendo uma visão clara sobre faturamento, volume de vendas, produtos e clientes, com foco em **apoiar o dono do negócio na tomada de decisões**.

## 🗂️ Fonte dos Dados
Os dados foram obtidos a partir de PDFs, anotações manuais e controles informais utilizados no dia a dia da empresa.  
Essas informações foram organizadas e estruturadas em planilhas Excel para possibilitar a análise.

## 🔄 Processo ETL
- Organização dos dados em tabelas de clientes, produtos, vendas e itens de venda  
- Limpeza e padronização dos dados no Power Query  
- Modelagem simples com tabela fato e dimensões  
- Criação de medidas e KPIs utilizando DAX  

## 📈 Indicadores Analisados
- 💰 Faturamento Total  
- 🛒 Quantidade de Vendas  
- 🎟️ Ticket Médio  
- 📦 Quantidade de Produtos Vendidos  
- 👤 Clientes Ativos  
- 📊 Crescimento Percentual  

## 📊 Visualizações
- 📈 Evolução do faturamento ao longo do tempo  
- 💰 Faturamento por produto  
- 📦 Quantidade vendida por produto  
- ⭐ Principais clientes por faturamento  

## 🛠️ Ferramentas Utilizadas
- Power BI  
- Excel  
- Power Query  
- DAX  
- Figma (layout do dashboard)  

## 📌 Uso do Dashboard
O dashboard foi desenvolvido para **uso prático no dia a dia do negócio**, ajudando o **proprietário** a:
- Entender se o faturamento está crescendo ou caindo  
- Identificar quais produtos geram mais receita  
- Comparar volume de vendas com valor faturado  
- Acompanhar os clientes que mais contribuem para o faturamento  

## 🔄 Manutenção e Atualizações
O dashboard pode receber **atualizações conforme novos dados forem registrados**, além de ajustes simples em indicadores e visualizações, de acordo com a necessidade do negócio.

## ✅ Resultados
A análise trouxe maior clareza sobre o desempenho das vendas, mostrando que o negócio trabalha com **baixo volume e alto valor por venda**, facilitando decisões mais conscientes sobre produtos e clientes.
