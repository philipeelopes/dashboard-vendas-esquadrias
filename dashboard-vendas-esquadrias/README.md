# 📊 Dashboard de Vendas – Empresa de Esquadrias

## 🎯 Objetivo
Este projeto tem como objetivo analisar o desempenho de vendas de uma empresa de esquadrias de alumínio, trazendo visibilidade sobre faturamento, volume de vendas, produtos e clientes, apoiando a tomada de decisão baseada em dados.

## 🗂️ Fonte dos Dados
Os dados foram coletados a partir de PDFs, anotações manuais e controles informais da empresa, sendo estruturados e organizados em planilhas Excel para análise.

## 🔄 Processo ETL
- Organização dos dados em tabelas de clientes, produtos, vendas e itens de venda
- Limpeza e padronização dos dados no Power Query
- Modelagem de dados com tabela fato e dimensões
- Criação de medidas e KPIs utilizando DAX

## 📈 Indicadores Analisados
- Faturamento Total
- Quantidade de Vendas
- Ticket Médio
- Quantidade de Produtos Vendidos
- Clientes Ativos
- Crescimento percentual

## 📊 Visualizações
- Evolução do faturamento ao longo do tempo
- Faturamento por produto
- Quantidade vendida por produto
- Top clientes por faturamento

## 🛠️ Ferramentas Utilizadas
- Power BI
- Excel
- Power Query
- DAX
- Figma (layout do dashboard)

## ✅ Resultados
A análise permitiu maior clareza sobre o desempenho da empresa, identificando diferenças entre volume de vendas e faturamento, além de apoiar decisões estratégicas relacionadas a produtos e clientes.
